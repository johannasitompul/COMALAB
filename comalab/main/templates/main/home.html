Hello, there!

Covid-19 Machine Learning Application or COMALAB in short is a web application that accurately predicts the risk of someone having contracted the corona virus through the x-ray images of their chest. The machine learning model built on Convolutional Neural Network (CNN) is trained using more than six thousand x-ray images and a technique termed transfer learning. With that, we were able to reach a prediction accuracy of 94%!
